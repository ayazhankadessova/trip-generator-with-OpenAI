## Run Locally

Clone the project

```bash
  git clone https://github.com/ayazhankadessova/trip-generator-OpenAI.git
```

Go to the project directory

```bash
  cd ..trip-generator-OpenAI
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
